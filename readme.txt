Реализовано подключение по публичному API к биржам HitBTC и Poloniex. Считываются OrderBooks и берутся лучшие цены покупки и продажи. В браузере отображается разница цен покупки на одной бирже и продажи на другой.
Для запуска необходимо:
1.Распаковать и импортировать maven-проект в среду разработки.
2.В корне проекта выполнить команду mvn jetty:run
3.В браузере перейти по ссылке localhost:8080/currency
4.Выбрать валюты для сравнения (Bitcoin/Dollar или Ethereum/Bitcoin) и отправить запрос на исполнение.
Клиент будет перенаправлен на страницу с отображением лучшей цены купли/продажи на биржах HitBTC и Poloniex. Частота обновления ордеров раз в 10 секунд.
